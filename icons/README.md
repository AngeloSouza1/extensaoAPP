# Ícones da Extensão

Esta pasta deve conter os seguintes arquivos de ícone:

- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels  
- `icon128.png` - 128x128 pixels

## Como criar os ícones

Você pode usar qualquer ferramenta de edição de imagens ou geradores online. Algumas sugestões:

1. **Favicon.io** - https://favicon.io/
2. **Canva** - https://www.canva.com/
3. **GIMP** ou **Photoshop** para edição manual

O ícone pode ser um relógio, um ponto de verificação, ou qualquer símbolo relacionado a ponto eletrônico.

## Ícone temporário

Se você quiser testar a extensão rapidamente, pode criar ícones simples usando um gerador online ou até mesmo usar imagens temporárias. A extensão funcionará mesmo sem os ícones, mas eles aparecerão como quebrados na interface do navegador.
